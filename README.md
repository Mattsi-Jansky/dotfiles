# Dotfiles

## Prerequisites

macOS or Debian-based distro (only tested on Pop!OS). Installs all dependencies automatically, with the exception of Alacritty for Linux.

## Setup

* `git clone git@github.com:Mattsi-Jansky/dotfiles.git ~/.dotfiles`
* `~/.dotfiles/apply.sh`
